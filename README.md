# kamidana-docker

## usage

image pull

```
docker pull ghcr.io/kukv/kamidata-cli:latest
```

or

```
docker pull ghcr.io/kukv/kamidata-cli:<<release tag>>
```

## execute

Just pass the kamidata-cli command at runtime.

see: https://github.com/podhmo/kamidana
